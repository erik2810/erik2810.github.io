DiffQFT: Differentiable Quantum Field Theory

A computational framework translating the Sine-Gordon Model in Hyperbolic Space directly into executable PyTorch code.

  • PyTorch
  • Monte Carlo Integration
  • AdS/CFT
  • FastAPI

In my thesis, I derived the anomalous dimensions of vertex operators in EAdS₂ analytically. DiffQFT automates this by:

  • Simulating Witten Diagrams (4-point functions) via Monte Carlo.
  • Using an Active Learning loop to train a Neural Surrogate.
  • Solving the Inverse Holography problem: recovering bulk potentials from boundary data.
